# 📘 中考错题本

> 一个帮助中考学生按科目整理错题的轻量级 Web 工具。

## ✨ 功能

- **按科目分类** — 数学、物理、化学、英语、语文等科目独立管理
- **上传图片** — 支持拍照上传错题原题，不用手抄
- **记录错因** — 记录错误原因和涉及的知识点
- **复习状态** — 跟踪每道错题是否已掌握
- **本地存储** — 数据保存在浏览器，无需注册

## 🛠️ 技术栈

HTML + CSS + Vanilla JavaScript（纯前端，无需服务器）

## 🎯 适用场景

- 中考备战期间的错题整理
- 各科目错题分类复习
- 考前冲刺查漏补缺

## 🚀 快速开始

直接用浏览器打开 `index.html` 即可使用。
